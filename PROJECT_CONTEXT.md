# Inventory App

Purpose:
Inventory and sales management application.

Current goal:
Create a premium public catalog showing products currently in stock.

Target audience:
Retail customers.

Main requirement:
The public website is not an admin panel.
It should look like a premium product showcase.

Key principles:

* Real products only.
* Show stock availability.
* Show price immediately.
* Strong WhatsApp/Telegram CTA.
* Mobile-first experience.
* Fast loading.
* Premium appearance.

Tech stack:

* Node.js
* Express
* Vanilla JavaScript
* HTML/CSS
